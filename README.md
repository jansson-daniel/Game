# Slot-machine Game

Author: Daniel Jansson
Date: 29/8-17
Description: A casino slot-machine game.

Tested on:
  - desktop:
    Chrome
    Safari
    Firefox
    IE11

  - mobile:
    Android and IOS.

Techniques:
    JavaScript
    Node.js (express)
    Webpack
    Babel
    NPM
    ES6

Setup:
1. npm install
2. npm run start
