# Slot-machine Game

Author: Daniel Jansson

Date: 29/8-17

Description: A casino slot-machine game.

Tested on:
  - desktop:
    Chrome
    Safari
    Firefox
    IE11

  - mobile:
    Android and IOS.

Techniques:
    JavaScript
    Node.js (express)
    Webpack
    Babel
    NPM
    ES6

Installation:
 - npm install
 - npm run start
 - Runs on port: 8080

Instructions:
  Click button to play. Three different outcomes: no-win, small-win and big-win.

  If small-win or big-win, bonus spin might occur if you are lucky.
